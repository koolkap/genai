1. Setup the Python by installing the Windows 

https://www.python.org/downloads/

2. Open the visual code and go to 2-Session run the command

python -m venv venv 

to create virtual environment so that whatever packages you install remain to this repo and not conflict with other repo packages

3. Activate the venv using the simple activate command

.\venv\Scripts\activate

4. Create main.py file, check your python environment is setup correctly by Simple "Hello World"

python .\main.py
