1. Setup the Python by installing the Windows 

https://www.python.org/downloads/

2. Open the visual code and go to 2-Session run the command

python -m venv venv 

to create virtual environment so that whatever packages you install remain to this repo and not conflict with other repo packages

3. Activate the venv using the simple activate command

.\venv\Scripts\activate

4. Create main.py file, check your python environment is setup correctly by Simple "Hello World"

python .\main.py

4.1 Read the documentation of how to use the open AI platform 

https://platform.openai.com/docs/overview

5. Learn about the token usage with the model

https://platform.openai.com/settings/organization/limits

6. Free user credits and usage details

https://platform.openai.com/settings/organization/usage

7. Creating the new API key for your project

https://platform.openai.com/settings/organization/api-keys

8. Using the postman to talk to your API

https://api.openai.com/v1/models

Add to header Authorization: Bearer <your token>

9. Now run small python code in main.py to talk to Open AI API
